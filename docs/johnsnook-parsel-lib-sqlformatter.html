Class johnsnook\parsel\lib\SqlFormatter

All Classes | Properties | Methods | Constants
Inheritancejohnsnook\parsel\lib\SqlFormatter

SQL Formatter is a collection of utilities for debugging SQL queries.

It includes methods for formatting, syntax highlighting, removing comments, etc. It's been heavily modified from the original by pimcore

See also https://github.com/pimcore/search-query-parser.
